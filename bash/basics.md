Conditional, if then else else if

```shell
if [ "dog" = "dog" ]; then
  echo "Dooooooooggg!! Doooooogggggg!"
fi

dog="lassie"
if [ "$dog" = "lassie"]; then
  echo "Timmy in the Well!"
else
  echo Timmy is fine
fi

wolf="lobo"
if [ "$wolf" = "fenrir" ]; then
  echo Destroy Asgard
elif [ "$wolf" = "lobo" ]; then
  echo "awoooooooooooo"
else
  echo "must just be a dog
fi
```
